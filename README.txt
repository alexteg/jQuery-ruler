Dependencies:

Lightweight (1.31 kB minified + 1 kB CSS) jQuery pixelruler plugin for measuring the pixel size of anything on a webpage (absolutely positioned). Currently horizontal only, with customizable width, draggable and resizable between a min- and max-width. Either resizable by clicking and dragging the right side or auto-resize with the mouse.

See demo.html for example usage
